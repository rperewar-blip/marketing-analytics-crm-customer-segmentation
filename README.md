# Marketing Analytics & CRM Customer Segmentation

## 📌 Project Overview
This project analyzes CRM data to evaluate customer behavior, segment customers, and measure marketing campaign effectiveness.

## 🎯 Objectives
- Segment customers based on spending patterns
- Analyze campaign performance metrics
- Identify high-value customers
- Evaluate churn risk indicators

## 🛠 Tools Used
- Excel / Python
- Data Visualization
- Statistical Analysis
- Customer Segmentation Techniques

## 📊 Key Insights
- 20% of customers contribute to majority of revenue
- Targeted campaigns show higher conversion rate
- Certain segments show higher churn probability

## 📁 Files Included
- CRM Dataset
- Analysis Files
- Visualizations
- Business Recommendations
